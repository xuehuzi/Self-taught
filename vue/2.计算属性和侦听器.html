方式1：模板内表达式

{{ message.split('').reverse().join('') }}

上面内容是直接在插值语法内写逻辑{{ message.split('').reverse().join('') }}
方式2：computed内计算属性

正常: "{{ message }}"

翻转: "{{ reversedMessage }}"

这是在computed:{reversedMessage: function () {return this.message.split('').reverse().join('')}}
方式3：通过方法实现

{{message}}

{{reversedMessage()}}

翻转绑定的是{{reversedMessage()}}，methods内调用reversedMessage()方法实现
依赖其他计算属性
{{text}}
{{reversedMessage}}

两个vue实例app3和app4,在app4内计算属性依赖的是app3的数据text
